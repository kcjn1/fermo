## EN

### Catalog Title

Fermo

### Short Description

Local-first focus contracts for macOS.

### Description

Fermo helps you start one protected work session with a clear task, allowed tools, blocked distractions, and a local evidence log at the end. It is a native macOS utility built around Focus Rooms, Blocklists, schedules, and break-glass transparency.

Fermo runs locally. It uses macOS system permissions to apply website and app rules during active sessions. It does not use cloud sync or paid AI in this version.

### Privacy Copy

Fermo stores rooms, sessions, schedules, preferences, diagnostics, and evidence logs locally on your Mac. Evidence export writes Markdown files only to the folder you choose. Fermo does not upload your focus history, evidence, app rules, or browsing rules to Toolary servers in this version.

Fermo requires macOS permissions for system extensions, network filtering, helper login items, and app launch enforcement. These permissions are used only to apply the focus contract you start or schedule.

### Permission Copy

Fermo needs macOS approval before it can protect a session. Network Extension approval enables website blocking. Endpoint Security approval enables app launch enforcement. Login Item approval lets the helper restore active sessions after relaunch or login.

Fermo pomaga rozpocząć jedną chronioną sesję pracy z jasnym zadaniem, dozwolonymi narzędziami, zablokowanymi rozpraszaczami i lokalnym dziennikiem dowodów na końcu. To natywne narzędzie macOS oparte na Focus Rooms, Blocklists, harmonogramach i przejrzystym break-glass.

### Prywatnosc

Fermo przechowuje pokoje, sesje, harmonogramy, preferencje, diagnostykę i dziennik dowodów lokalnie na Twoim Macu. Eksport dowodów zapisuje pliki Markdown tylko w folderze, który wybierzesz. W tej wersji Fermo nie wysyła historii skupienia, dowodów, reguł aplikacji ani reguł stron na serwery Toolary.

Fermo wymaga uprawnień macOS dla system extensions, filtrowania sieci, helpera w Login Items i egzekwowania uruchamiania aplikacji. Te uprawnienia są używane tylko do stosowania kontraktu skupienia, który uruchomisz albo zaplanujesz.

### Uprawnienia

Fermo potrzebuje zgody macOS, zanim będzie mogło chronić sesję. Network Extension umożliwia blokowanie stron. Endpoint Security umożliwia egzekwowanie uruchamiania aplikacji. Login Item pozwala helperowi przywracać aktywne sesje po ponownym uruchomieniu aplikacji albo zalogowaniu.

## DE

### Katalogtitel

Fermo

### Kurzbeschreibung

Lokale Fokusvertraege fuer macOS.

### Beschreibung

Fermo hilft dir, eine geschuetzte Arbeitssitzung mit klarer Aufgabe, erlaubten Werkzeugen, blockierten Ablenkungen und lokalem Evidence Log zu starten. Es ist eine native macOS App mit Focus Rooms, Blocklists, Zeitplaenen und transparentem Break-glass.

Fermo laeuft lokal. Es nutzt macOS-Berechtigungen, um Website- und App-Regeln waehrend aktiver Sitzungen anzuwenden. Diese Version enthaelt keinen Cloud Sync und keine bezahlte KI.

### Datenschutz

Fermo speichert Raeume, Sitzungen, Zeitplaene, Einstellungen, Diagnosen und Evidence Logs lokal auf deinem Mac. Der Evidence Export schreibt Markdown-Dateien nur in den Ordner, den du auswaehlst. In dieser Version laedt Fermo deine Fokus-Historie, Nachweise, App-Regeln oder Website-Regeln nicht auf Toolary-Server hoch.

Fermo benoetigt macOS-Berechtigungen fuer Systemerweiterungen, Netzwerkfilterung, Login-Item-Helper und App-Launch-Enforcement. Diese Berechtigungen werden nur genutzt, um den Fokusvertrag umzusetzen, den du startest oder planst.

### Berechtigungen

Fermo benoetigt macOS-Freigaben, bevor eine Sitzung geschuetzt werden kann. Die Network Extension aktiviert Website-Blocking. Endpoint Security aktiviert App-Launch-Enforcement. Das Login Item erlaubt dem Helper, aktive Sitzungen nach App-Neustart oder Login wiederherzustellen.
